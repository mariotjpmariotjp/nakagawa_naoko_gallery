# 中川直子さん専用 2Dギャラリー（ラピスラズリ版）

## ファイル
- `nakagawa_naoko_gallery.html`：画像込みの単一HTMLです。この1ファイルだけで動きます。

## 公開URL例
GitHub Pages の `gallery` リポジトリにアップロードした場合：

```text
https://mariotjpmariotjp.github.io/gallery/nakagawa_naoko_gallery.html
```

はてなブログからは、このURLへリンクを貼ってください。

## 今回入れた機能
- ラピスラズリの濃淡を基調にした幻想的なデザイン
- 作品一覧と、一点ずつじっくり見る詳細モーダルの2段構え
- テーマ、サイズ、飾る場所、価格帯、並び順による絞り込み
- 各作品に「サイズ」「飾る場所のヒント」「価格目安」「作家コメント見本」を表示
- 複数作品を飾る時のティップ掲載
- 作家へのコンタクトリンク

## 作品データの差し替え場所
HTML末尾の以下の配列を編集します。

```js
const works = [ ... ];
```

各項目の意味：
- `title`：作品名
- `theme`：テーマ（例：窓辺と花、夜と月）
- `size`：サイズ分類（例：S・縦長、M・横長）
- `area`：並び替え用の面積目安
- `place`：飾る場所（例：廊下、寝室、テラス）
- `price`：絞り込み用の数値価格
- `priceLabel`：画面に出す価格目安
- `hint`：飾る場所の一言ヒント
- `comment`：作家コメント欄。中川さんご本人の文章に差し替えます

## 連絡先リンクの変更
HTML内の以下を実際の連絡先に差し替えてください。

```html
mailto:contact@example.com
```

メール以外にも、Googleフォーム、Instagram、はてなブログのお問い合わせページURLに変更できます。
